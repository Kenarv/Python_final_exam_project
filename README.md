-----Cubix Python képzés vizsga munka-----

A program a képzés során példaként használt Quiz feladat kiegészitett verziója.

- Ez kiegészített verzió rendelezik három kérdés kategóriával , így a felhasználó válogathat különböző témákból.
- Valamint a program bekér játékos nevet, ami alapján képes menteni a végeredményt és azt a quiz végén rangsorban kilistázni.
- A program valamennyi elemet tartalmaz amik a képzés során felmerültek.
